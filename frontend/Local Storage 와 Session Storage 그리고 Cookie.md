# Local Storage 와 Session Storage 그리고 Cookie

- 모두 클라이언트 상에서 Key / Value 페어를 저장할 수 있는 메커니즘으로 value는 반드시 문자열이어야 한다. 모두 동일 출처 정책 (SOP)을 따르기 때문에, 다른 도메인에서 접근 할 수 없다.

|               | cookie           | local storage           | session storage          |
| ------------- | ---------------- | ----------------------- | ------------------------ |
| 생성자        | 클라이언트/서버  | 클라이언트              | 클라이언트               |
| 지속시간      | 설정 여부에 따름 | 명시적으로 지울 때 까지 | 탭 / 윈도우 닫을 때 까지 |
| 서버와의 통신 | O                | X                       | X                        |
| 용량          | 5KB              | 5MB/10MB                | 5MB                      |
| 취약점        | XSS/CSRF 공격    | XSS 공격                | XSS 공격                 |
