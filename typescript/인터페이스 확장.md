## 인터페이스 확장

- 기존에 있는 인터페이스를 상속받아서 확장하는 것

```typescript
interface Person {
    name: string;
    age : number;
}

interface Developer extends Person {
    langauge:string
}

let me : Developer = {
    name : 'moonee',
    age:'24',
    langauge:'typescript'
}
```

- 인터페이스를 정의 할 때 이전에 이미 정의해놓은 인터페이스와 겹치는 속성이 있을 때 사용 할 수 있다.
